# Descripción del proyecto
Desafío del curso Python para Data Science.  El objetivo es identificar la tienda menos rentable y dar una recomendación basada en los datos.

**Lista de tecnologías**
* **Lenguaje de Programación:** Python
* **Análisis y Manipulación de Datos:** Pandas
* **Visualización de Datos:** Matplotlib
* **Entorno de Desarrollo:** Jupyter (usando Google Colab)

**Como obtener una copia del repositorio**
Paso 1: Dirígete a la URL, selecciona el botón verde 'Code' y después 'Download ZIP'
Paso 2: Ve a tu carpeta local y usa el siguiente comando en tu terminal:
git clone https://github.com/wensguillen/analisis_ventas_tiendas
No te olvides de instalar las librerías necesarias para trabajar con el repositorio:
pip install -r requirements.txt
